# Banking System (C++)

A console-based banking system built in C++ to practice OOP, file handling, and permission-based access.

## Features
- User login
- Permission-based access (users & roles)
- Client management: Add / Update / Delete / Find
- Transactions: Deposit / Withdraw / Transfer (if included)
- Data stored locally in text files
- Menu-driven console interface

## Tech Stack
- C++
- OOP
- File handling (read/write)
- Console UI

## How to Run (Visual Studio)
1. Open `Bank System.sln`
2. Build the solution
3. Run the program and follow the menu
